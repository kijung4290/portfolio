# 사회복지사 포트폴리오 (Social Worker Portfolio)

안녕하세요! 이 프로젝트는 코딩 없이 관리할 수 있는 포트폴리오 사이트입니다.

## 🚀 시작하는 방법

1. **터미널 열기**: VS Code에서 `Ctrl + ~`를 눌러 터미널을 엽니다.
2. **실행 명령어 입력**: 아래 명령어를 입력하고 엔터를 누르세요.
   ```bash
   npm run dev
   ```
3. **사이트 접속**: 브라우저 주소창에 `http://localhost:3000`을 입력하면 포트폴리오가 보입니다.

## ✏️ 내용 수정하는 방법 (관리자 페이지)

1. 주소창에 `http://localhost:3000/admin`을 입력하여 접속하세요.
2. **비밀번호**: `admin1234`
3. 로그인 후, 화면에서 내용을 직접 수정하고 **[변경사항 저장]** 버튼을 누르세요.
4. 메인 페이지로 돌아가면 수정된 내용이 즉시 반영됩니다!

## 📸 사진 넣는 방법

1. `public/images` 폴더에 원하는 사진 파일을 넣으세요. (예: `profile.jpg`)
2. 관리자 페이지나 `data/portfolio.json` 파일에서 이미지 경로를 `/images/profile.jpg`와 같이 입력하세요.

## 🛠️ 고급 설정 (Supabase 연동 시)

이 프로젝트는 현재 **로컬 파일(data/portfolio.json)**을 데이터베이스처럼 사용하고 있습니다. 
나중에 인터넷에 배포(Vercel 등)하려면 Supabase 데이터베이스 연결이 필요할 수 있습니다. 
그때는 `lib/supabase.js` 등을 설정하여 연동할 수 있습니다.

문의사항이 있으시면 언제든지 물어보세요!
